# Vlither Enhanced 1.2.0

## The Control Evolution Release

**Released:** 19 July 2026  
**Created, engineered and maintained by:** **OM Rajput**  
**Android version code:** `10203`

Vlither Enhanced 1.2.0 turns the mobile client into a considerably more
personal arena instrument. This release expands the control system beyond a
single input style, introduces server discovery directly inside the client,
and refines the everyday Android interactions that decide whether an app feels
merely functional or genuinely native.

The result is the most complete Vlither Enhanced build so far: faster to enter,
easier to configure, more expressive to control and substantially more
comfortable across different hands, screens and play styles.

## A new control language: Arrow Steering

- Introduces **Arrow Steering** as a full control mode alongside the existing
  joystick system.
- Keeps direction control relative to the snake instead of forcing the player
  to steer toward the exact point where the finger first touches the display.
- Allows direction to be adjusted fluidly from the active touch area while the
  arrow remains visually connected to the snake's heading.
- Adds independent arrow distance, scale and steering-smoothness adjustment.
- Adds a visual colour palette so players can choose an arrow colour directly
  instead of entering technical colour values.
- Adds an optional animated boost emphasis that illuminates the arrow itself
  without covering the arena with an unnecessary ring.
- Supports a second-finger boost gesture while the primary finger continues to
  steer.
- Presents a live preview beside the Arrow settings so visual changes can be
  understood before returning to the arena.

Arrow Steering is not a cosmetic variation of the joystick. It is a separate,
purpose-built input style for players who prefer direct directional movement
and a clearer relationship between touch motion and snake heading.

## Joystick and boost, properly separated

- Gives Joystick and Arrow controls their own dedicated configuration areas.
- Strengthens touch ownership so the joystick region remains steering-only and
  does not accidentally activate boost.
- Keeps the boost region dedicated to boost behavior instead of allowing it to
  interfere with direction input.
- Retains floating and fixed joystick modes, fixed boost positioning and
  handedness-aware layouts.
- Refines the control-position editor so movable controls can be arranged for
  the player's grip rather than a one-size-fits-all screen layout.
- Preserves independent size, opacity and placement controls for the broader
  mobile control suite.

This separation matters most during fast play: each finger now has a clearer
job, and each control mode has settings that actually belong to it.

## Server selection inside the client

- Replaces the single raw server field with a dedicated **Select Server**
  journey.
- Adds a browsable server list designed for touch rather than desktop scroll
  bars.
- Adds server search for quickly narrowing a larger list.
- Keeps manual server entry available for private, testing or community-hosted
  endpoints.
- Allows a custom address to be saved and selected again without repeatedly
  typing it.
- Displays the chosen endpoint clearly on the Home screen before play begins.

Players can now move between discovered and custom servers without leaving
Vlither Enhanced or treating an IP address field as the entire server browser.

## Android interaction polish

- Enables natural finger-drag scrolling across server lists, Settings and
  other scrollable client surfaces.
- Retains visible scroll bars for position awareness while no longer requiring
  the player to drag the narrow bar itself.
- Adds long-press clipboard paste support to relevant text-entry fields.
- Refines slider widths and supporting labels so controls remain readable on
  smaller landscape displays.
- Improves touch targets and spacing throughout the expanded control menus.
- Reworks the launcher-icon composition around Android's adaptive safe area so
  the Vlither Enhanced mark sits more cleanly across different launchers.

These refinements make configuration feel like part of an Android application,
not a desktop settings panel viewed through a phone.

## The visual system grows with the feature set

- Extends the graphite, silver and emerald interface language across Server
  Selection and the expanded Controls experience.
- Preserves the focused Home command panel, metallic Vlither Enhanced identity
  and arena-inspired grid atmosphere.
- Keeps the renderer-driven Skin Editor workflow intact while the surrounding
  application continues to evolve.
- Maintains clear hierarchy between primary actions, configuration tools,
  previews and supporting information.

Version 1.2.0 adds substantially more capability without turning the interface
into a crowded collection of unrelated panels.

## Before installing

- This build is intended for 64-bit ARM Android devices with Vulkan support.
- Existing official users should install it over the current app to preserve
  the normal Android update path.
- Download only `Vlither-Enhanced-v1.2.0.apk` from the official v1.2.0 release.
- Android may ask the browser or file manager for permission to install an APK
  distributed outside the Play Store.
- Device firmware and vendor GPU drivers differ; compatibility reports should
  include the phone model, Android version and GPU name.

## A platform ready for what comes next

Version 1.1.0 established the foundation. Version 1.2.0 makes that foundation
meaningfully playable: multiple control philosophies, a real server-selection
flow and a more mature Android interaction layer now live inside one coherent
client.

Future releases can extend the control catalogue and server experience without
rebuilding the application around them. The architecture is becoming a
platform for Vlither Enhanced rather than a sequence of disconnected
experiments.
